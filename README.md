# LeetCode-SQL-Solutions
🚀 SQL Interview Preparation (Solved with PostgreSQL)
📌 Overview

This repository contains fully solved SQL interview questions across three difficulty levels, along with an implemented study plan.

All problems are solved using PostgreSQL, with a focus on writing clean, efficient, and production-ready queries.
